# webs-2026sjb-guide

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面各自独立，不隶属于任何特定网站或域名，主要用于保存和展示静态内容。

## 项目简介

本仓库收集并整理了若干 HTML 文件，每个文件都是一个完整的独立页面。适用于需要长期保存的静态网页素材、示例页面或小型文档页面。仓库本身不提供任何在线服务，仅作为版本管理和文件存档使用。

## 目录说明

```
.
├── README.md          # 本说明文件
├── pages/             # 存放主要 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 页面引用的资源文件（如图片、样式、脚本）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下存放所有归档的 HTML 页面。每个文件均可通过直接打开或部署后访问。
- `assets/` 目录用于存储页面所需的公共资源，按类型分文件夹存放。

## 页面归档说明

- 每个 HTML 页面均为自包含的独立文件，部分页面可能引用 `assets/` 目录下的资源。
- 所有页面均经过基础整理，确保在主流浏览器中正常显示。
- 归档内容可能包括示例、演示、资料页等，不涉及任何特定商业或推广用途。
- 页面内容可能会随时间更新或补充，具体变更可查看提交历史。
